# WanderGo

WanderGo is a simple travel website built using **HTML** and **CSS**. It was created as part of my Full Stack Development learning journey to practice front-end web development concepts and improve my skills in designing visually appealing, responsive, and user-friendly websites.

## Features

- Responsive homepage layout
- Modern navigation bar with search section
- Popular travel destinations
- Featured hotels and restaurants
- Food and travel experience sections
- Attractive hover effects and interactive UI
- Beautiful footer section
- Clean and modern travel-themed design

## Technologies Used

- HTML5
- CSS3

## What I Learned

- Structuring multi-section webpages using semantic HTML
- Styling responsive layouts with CSS
- Working with Flexbox for page layouts
- Creating reusable UI components
- Applying hover effects and transitions
- Improving website design, spacing, and visual hierarchy

## Purpose

The goal of this project was to recreate the look and feel of a modern travel website while strengthening my front-end development skills through hands-on practice. This project helped me gain confidence in building responsive layouts and creating an engaging user experience using only HTML and CSS.

## Future Improvements

- Add JavaScript for interactive features
- Implement destination search functionality
- Add booking and reservation features
- Improve responsiveness for all screen sizes
- Integrate travel APIs for real-time destination and hotel information
- Add user authentication and personalized travel recommendations

This project demonstrates my ability to design and develop a multi-section travel website using core front-end technologies while focusing on clean code, responsive layouts, and an attractive user interface.
